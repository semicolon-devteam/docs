## 기능 스펙 — 주문 플로우

## 기능 스펙: 주문 플로우

### 1. 메뉴 탐색
- 진입 시 카테고리 탭 노출 (기본: "전체")
- 카테고리 선택 시 해당 메뉴만 필터
- 메뉴 카드: 썸네일 + 이름 + 가격
- 품절 메뉴는 하단 배치 + 품절 오버레이

### 2. 옵션 선택
- 메뉴 탭 → 바텀시트 오픈
- 옵션 그룹별 라디오/체크 선택
  - 사이즈: R(기본) / L(+500)
  - 샷: 기본 / +1샷(+500)
  - 시럽: 없음 / 바닐라 / 헤이즐넛
- 수량 선택 (기본 1, 최대 10)
- "장바구니 담기" CTA → 토스트 알림

### 3. 장바구니
- 담긴 아이템 목록 (메뉴명, 옵션, 수량, 소계)
- 수량 변경 스테퍼
- 스와이프 삭제
- 합계 금액 실시간 계산
- "주문하기" CTA

### 4. 결제
- 닉네임 입력 (2~6자, 한글/영문)
- 결제 수단: 카카오페이
- 카카오페이 SDK → 결제 완료 콜백
- 실패 시 재시도 안내

### 5. 주문 완료
- 대기번호 표시
- 예상 준비시간 (평균 5분)
- 주문 상태 실시간 표시

### 예외 처리
- 결제 중 품절 전환 → "선택하신 메뉴가 품절되었습니다" 알림
- 영업시간 외 접속 → 주문 버튼 비활성 + "영업 준비 중" 안내

## 기능 스펙 — 관리자 대시보드

## 기능 스펙: 관리자 대시보드

### 1. 주문 관리 화면
- 실시간 주문 카드 목록 (최신순)
- 카드: 주문번호, 닉네임, 메뉴 목록, 접수 시간
- 액션 버튼: "접수" / "완료" / "거절"
- 접수 후 3분 경과 시 카드 테두리 경고색
- 거절 시 사유 선택 (재료 소진 / 영업 종료 / 기타)

### 2. 메뉴 관리
- 메뉴 목록 (카테고리별 그룹)
- 등록: 사진 업로드 + 이름 + 가격 + 옵션 그룹 설정
- 수정: 인라인 편집 (가격, 품절 토글)
- 삭제: 소프트 삭제 (주문 이력 보존)
- 카테고리 순서 드래그 정렬

### 3. 매출 요약
- 오늘 매출 (건수 + 금액)
- 주간 추이 그래프 (막대 차트)
- 인기 메뉴 TOP 5

### 4. 설정
- 영업시간 설정 (요일별)
- 카페 이름/로고 변경
- 알림 수신 설정
