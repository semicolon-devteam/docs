## 명확화 Q&A

### Q1. 메뉴 옵션 구조
**Q:** 메뉴 옵션(사이즈, 샷 추가 등)은 어떤 구조인가요?
**A:** 사이즈(R/L), 샷 추가(+500원), 시럽 변경 정도. 옵션 그룹은 최대 3개면 충분합니다.

### Q2. 결제 수단 범위
**Q:** 카카오페이 외 다른 결제 수단이 필요한가요?
**A:** MVP는 카카오페이만. 런칭 후 네이버페이 추가 예정.

### Q3. 주문 접수 방식
**Q:** 주문이 들어오면 자동 접수인가요, 사장이 수동 수락하나요?
**A:** 수동 수락. 재료 소진 시 거절할 수 있어야 합니다. 3분 내 미응답 시 자동 알림.

### Q4. 영업시간 외 주문
**Q:** 영업시간 외에 주문이 가능한가요?
**A:** 아닙니다. 영업시간 설정 기능이 필요하고, 시간 외에는 "영업 준비 중" 표시.

### Q5. 테이블 번호 / 픽업 구분
**Q:** 매장 내 테이블 번호 입력이 필요한가요?
**A:** 아니요. 카운터 픽업 전용이라 닉네임만 입력받으면 됩니다.
