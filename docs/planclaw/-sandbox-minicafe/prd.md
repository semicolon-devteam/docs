## PRD (제품 요구사항 정의서)

## MiniCafe PRD

### 1. 제품 비전
카페 고객이 줄 서지 않고 1분 내 주문을 완료할 수 있는 모바일 웹앱.

### 2. 사용자 페르소나
- **단골 고객 수진**: 30대 직장인, 출근길에 매일 아메리카노 주문. 빠른 재주문이 핵심.
- **카페 사장 민아**: 비전공자, 메뉴 관리와 매출 확인이 편해야 함.

### 3. 핵심 기능 (MVP)
| 우선순위 | 기능 | 설명 |
|---------|------|------|
| P0 | 메뉴 조회 | 카테고리, 사진, 가격, 옵션 |
| P0 | 주문 + 결제 | 장바구니 → 카카오페이 결제 |
| P0 | 주문 관리 | 사장용 주문 접수/완료 대시보드 |
| P1 | 재주문 | 최근 주문 1탭 재주문 |
| P1 | 매출 요약 | 일간/주간 매출 그래프 |
| P2 | 알림 | 주문 완료 시 고객 푸시 |

### 4. 제외 사항 (MVP)
- 배달 기능
- 멤버십/포인트
- 다국어 지원
