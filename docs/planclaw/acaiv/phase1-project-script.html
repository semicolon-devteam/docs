🎨 2. 화면 1: 프로젝트 생성

UI 컴포넌트

A. 비디오 포맷 선택

타입: Radio Button Group

옵션:

  • 16:9 (롱폼 유튜브) - 1920x1080
  • 9:16 (쇼츠, 릴스) - 1080x1920
  • 1:1 (인스타그램) - 1080x1080
  • 3:4 (세로형) - 1080x1440

기본값: 16:9

동작: 하나만 선택 가능, 선택 시 프리뷰 업데이트

B. 스타일 템플릿 선택

타입: Grid Gallery (3x3)

옵션:

  • 귀여운 졸라맨 (Stick Figure)
  • 일본 애니메이션 (Japanese Anime)
  • 미니멀 인포그래픽 (Minimal Infographic)
  • 3D 애니메이션 (3D Animation)
  • 카툰 해설 (Cartoon Commentary)
  • 영화 같은 스타일컷 (Cinematic)
  • 미국 카툰 (American Cartoon)
  • 플랫 일러스트 (Flat Illustration)
  • 사용자 정의 (Custom)

기본값: 귀여운 졸라맨

동작: 선택 시 스타일 프롬프트 자동 설정. "사용자 정의" 선택 시 텍스트 입력 필드 표시

C. 다음 단계 버튼

타입: Primary Button

레이블: "다음 단계 →"

동작: 프로젝트 생성 API 호출 → 2단계(대본 생성) 화면으로 이동

유효성 검사: 포맷과 스타일 모두 선택되어야 활성화

API 연동

POST /api/projects
Request:
{
  "format": "16:9",
  "style": "stick-figure",
  "stylePrompt": null
}

Response:
{
  "id": "clx...",
  "userId": "clx...",
  "format": "16:9",
  "style": "stick-figure",
  "status": "draft",
  "createdAt": "2026-03-10T..."
}

예외 처리

상황처리 방법
포맷 미선택버튼 비활성화
스타일 미선택버튼 비활성화

📝 3. 화면 2: 대본 생성

UI 컴포넌트

A. 입력 방식 선택

타입: Tab Navigation

옵션:

  • AI 자동 생성
  • 직접 작성

기본값: AI 자동 생성

동작: 탭 전환 시 입력 UI 변경

B-1. AI 생성 모드: 주제 입력

타입: Text Input + Slider

필드:

  • 주제 입력 (1줄, 최대 100자)
  • 영상 길이 슬라이더 (15초, 30초, 60초, 5분, 10분)

버튼: "AI 대본 생성하기"

동작: Gemini 3 Pro API 호출 → 로딩 표시 (5~10초) → 대본 출력

B-2. 직접 작성 모드: 텍스트 에디터

타입: Textarea (Multiline)

기능:

  • 실시간 글자 수 표시
  • 예상 음성 길이 표시 (글자 수 ÷ 4)
  • 최대 2,400자 (10분 영상 기준)
C. 대본 편집 영역

타입: Rich Text Editor

기능:

  • AI 생성 후 수정 가능
  • 실시간 글자 수 카운터
  • 예상 영상 길이 표시
D. 다음 단계 버튼

레이블: "다음 단계 (음성 생성) →"

동작: 대본 저장 → 3단계(음성 생성) 화면으로 이동

유효성 검사: 대본이 10자 이상이어야 활성화

API 연동

POST /api/generate-script
Request:
{
  "topic": "환율 1500원 넘어가면 우리나라에 벌어지는 일들",
  "duration_seconds": 40,
  "style": "informative"
}

Response:
{
  "script": "환율 1,500원 시대. 우리 경제 진짜 비상등이 켜졌습니다...",
  "estimated_duration": 42,
  "char_count": 250,
  "creditsUsed": 5
}

PUT /api/projects/{projectId}
Request:
{
  "script": "환율 1,500원 시대...",
  "duration": 42
}

Response:
{
  "id": "clx...",
  "script": "환율 1,500원 시대...",
  "duration": 42,
  "updatedAt": "2026-03-10T..."
}

예외 처리

상황처리 방법
주제 미입력 (AI 모드)"주제를 입력해주세요" 알림
대본 10자 미만버튼 비활성화
API 타임아웃"잠시 후 다시 시도해주세요" + 재시도 버튼
Gemini API 에러"AI 생성 실패. 직접 작성을 시도해주세요"

🔄 4. 사용자 플로우

Step 1: 사용자가 "새 프로젝트" 클릭 → 화면 1 진입
Step 2: 비디오 포맷 선택 (16:9, 9:16, 1:1, 3:4 중 택1)
Step 3: 스타일 템플릿 선택 (9개 프리셋 + 커스텀 중 택1)
Step 4: "다음 단계" 클릭 → POST /api/projects → 프로젝트 생성
Step 5: 화면 2 진입 (projectId 전달)
Step 6: 입력 방식 선택 (AI 생성 or 직접 작성)
Step 7-A (AI 모드): 주제 입력 + 길이 선택 → "AI 대본 생성" 클릭 → POST /api/generate-script
Step 7-B (직접 작성): 텍스트 에디터에 대본 입력
Step 8: 대본 확인 및 수정 (선택)
Step 9: "다음 단계" 클릭 → PUT /api/projects/{projectId} → Phase 2 진입

💾 5. 데이터 모델

model Project {
  id            String   @id @default(cuid())
  userId        String
  name          String?
  format        String   // "16:9", "9:16", "1:1", "3:4"
  style         String   // "stick-figure", "anime", etc.
  stylePrompt   String?  // 사용자 정의 프롬프트
  characterSheet String? // 캐릭터 시트 URL (MVP 제외)
  script        String
  duration      Int      // 초 단위
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
  
  user          User     @relation(fields: [userId], references: [id])
  audio         Audio?
  segments      Segment[]
}
