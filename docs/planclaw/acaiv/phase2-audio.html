프로젝트: proj-acaiv | 이슈: #3

📱 화면 구성

A. 보이스 선택 섹션
- 150+ AI 보이스 라이브러리 (ElevenLabs)
- 보이스 카드: 이름, 언어, 성별, 샘플 재생 버튼
- 검색/필터 (언어, 성별, 음색)
B. 대본 미리보기
- Phase 1에서 작성한 대본 표시 (읽기 전용)
- 글자 수 & 예상 음성 길이 표시
C. TTS 생성 버튼
- "음성 생성하기" 버튼 (보이스 선택 후 활성화)
- 진행 상황 표시 (로딩 바 + 예상 소요 시간)
D. 오디오 플레이어
- 생성 완료 후 표시
- 재생/일시정지, 진행바, 음량 조절
- 다운로드 버튼 (선택)
