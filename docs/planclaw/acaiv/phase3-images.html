🖼️ Phase 3: 이미지 생성 (세그먼트)

프로젝트: proj-acaiv | 이슈: #4

📱 화면 구성

A. 자동 세그먼트 분할
- AI가 음성 길이 기반으로 컷 타이밍 자동 분석
- 세그먼트 목록 표시 (순서, 대본 텍스트, 타이밍)
- 세그먼트 추가/삭제/순서 변경 가능
B. 이미지 프롬프트 편집
- 각 세그먼트별 자동 생성된 이미지 프롬프트 표시
- 수동 편집 가능 (텍스트 입력)
- 스타일 일관성 프롬프트 자동 추가
C. 이미지 생성 & 미리보기
- "이미지 생성" 버튼 (세그먼트별 또는 전체)
- 진행 상황 표시 (생성 중인 세그먼트 번호)
- 생성된 이미지 썸네일 표시
- 재생성 버튼 (개별 세그먼트)
