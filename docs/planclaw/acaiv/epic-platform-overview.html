🤖 4. 오토파일럿 (핵심 기능)

원클릭 자동 생성: 주제 입력 + 설정만 하면 모든 단계가 자동으로 진행됩니다.

워크플로우

  1. 주제 입력
  2. 설정 (포맷, 스타일, 음성, 길이)
  3. 자동 생성 시작 → 모든 단계 자동 진행
  4. 실시간 진행 상황 UI (Socket.io)
  5. 완료 후 다운로드

예상 소요 시간

영상 길이 예상 소요 시간
40초 영상 약 11분
10분 영상 약 30~40분

실시간 진행 상황 UI

  • 현재 진행 중인 단계 표시 (1~6단계)
  • 각 단계별 진행률 (%) 표시
  • 예상 완료 시간 표시
  • 에러 발생 시 알림
  • 취소 버튼 (작업 중단 가능)

💰 5. 크레딧 시스템

내부용 간소화 버전

  • 관리자가 팀원별 크레딧 할당
  • 사용량 트래킹
  • 1 크레딧 = $0.01

10분 영상 1,500~2,000 크레딧 ~₩20,000~27,000

💾 6. 데이터 모델

User

model User {
  id            String    @id @default(cuid())
  email         String    @unique
  name          String
  credits       Int       @default(0)
  createdAt     DateTime  @default(now())
  updatedAt     DateTime  @updatedAt
  
  projects      Project[]
  creditTransactions CreditTransaction[]
}

Project

model Project {
  id            String   @id @default(cuid())
  userId        String
  name          String?
  format        String   // "16:9", "9:16", "1:1", "3:4"
  style         String   // "stick-figure", "anime", etc.
  stylePrompt   String?  // 사용자 정의 프롬프트
  script        String
  duration      Int      // 초 단위
  status        String   // "draft", "processing", "completed", "failed"
  creditsUsed   Int      @default(0)
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
  
  user          User     @relation(fields: [userId], references: [id])
  audio         Audio?
  segments      Segment[]
  video         Video?
}

Audio

model Audio {
  id            String   @id @default(cuid())
  projectId     String   @unique
  voiceId       String
  audioUrl      String
  duration      Int      // 초 단위
  creditsUsed   Int
  createdAt     DateTime @default(now())
  
  project       Project  @relation(fields: [projectId], references: [id])
}

Segment

model Segment {
  id            String   @id @default(cuid())
  projectId     String
  order         Int
  text          String
  imagePrompt   String
  imageUrl      String?
  videoUrl      String?
  startTime     Int      // 밀리초
  duration      Int      // 밀리초
  creditsUsed   Int      @default(0)
  createdAt     DateTime @default(now())
  
  project       Project  @relation(fields: [projectId], references: [id])
}

Video

model Video {
  id            String   @id @default(cuid())
  projectId     String   @unique
  videoUrl      String
  thumbnailUrl  String?
  srtUrl        String?
  duration      Int      // 초 단위
  resolution    String   // "1920x1080", "1080x1920", etc.
  creditsUsed   Int
  createdAt     DateTime @default(now())
  
  project       Project  @relation(fields: [projectId], references: [id])
}

CreditTransaction

model CreditTransaction {
  id            String   @id @default(cuid())
  userId        String
  amount        Int      // 양수: 충전, 음수: 사용
  type          String   // "allocation", "usage", "refund"
  description   String?
  createdAt     DateTime @default(now())
  
  user          User     @relation(fields: [userId], references: [id])
}
