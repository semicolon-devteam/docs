- **SAX-PO 에이전트 정리**: 개발자용 에이전트 4개를 SAX-Next로 이식

- `ddd-architect` agent: DDD 아키텍처
- `database-specialist` agent: DB 전문가

- `orchestrator`: 요청 라우팅
