# 3.26.0 (2025-11-27)

## Summary

docs 레포 `.claude/` 구조 플랫화 - sax-meta 중첩 제거

## Changed

### `.claude/` 구조 플랫화

`.claude/sax-meta/` 중첩 구조를 제거하고 `.claude/` 루트로 끌어올림.

**Before**:
```text
.claude/
├── CLAUDE.md           # 진입점 (패키지 선택)
├── sax-core/
└── sax-meta/           # 중첩 구조
    ├── CLAUDE.md       # SAX-Meta 상세 설정
    ├── agents/
    ├── skills/
    ├── scripts/
    └── templates/
```

**After**:
```text
.claude/
├── CLAUDE.md           # SAX-Meta 통합 설정
├── sax-core/           # Core 규칙 (유지)
├── agents/             # 끌어올림
├── skills/             # 끌어올림
├── scripts/            # 끌어올림
└── templates/          # 끌어올림
```

### CLAUDE.md 통합

- 2개의 CLAUDE.md (진입점 + 상세) → 1개로 통합
- 컨텍스트 로딩 효율성 향상

### 동기화 명령 변경

```bash
# Before
rsync -av --delete sax/packages/sax-meta/ .claude/sax-meta/

# After (플랫 구조)
rsync -av --delete \
  --exclude='sax-core' \
  --exclude='settings.local.json' \
  sax/packages/sax-meta/ .claude/
```

### [SAX]/[Semicolon AX] 키워드 규칙 제거

- docs 레포가 SAX-Meta 전용이므로 패키지 선택 키워드 불필요
- 시스템 메시지 접두사로서의 `[SAX]`는 유지

## Rationale

- docs 레포에서 SAX-Meta만 사용하므로 중첩 구조 불필요
- CLAUDE.md 2개 → 1개로 컨텍스트 효율성 향상
- 경로 단순화: `.claude/sax-meta/agents/` → `.claude/agents/`

## Migration Guide

### docs 레포지토리 사용자

변경 없음. 자동으로 적용됨.

### 동기화 스크립트 사용자

동기화 명령어가 변경됨:
```bash
rsync -av --delete \
  --exclude='sax-core' \
  --exclude='settings.local.json' \
  sax/packages/sax-meta/ .claude/
```
