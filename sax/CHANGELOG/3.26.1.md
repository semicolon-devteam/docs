## Migration

기존에 잘못 배포된 프로젝트는 다음 명령으로 정리:

```bash
# 잘못된 디렉토리 제거
rm -rf .claude/sax-next/
rm -rf .claude/sax-po/

# 재배포
./sax/scripts/deploy.sh sax-next /path/to/project --update
```
