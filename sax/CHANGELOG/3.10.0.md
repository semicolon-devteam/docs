# SAX v3.10.0 - SAX-Meta Orchestrator 및 Skill Creator Agent 추가

**Release Date**: 2025-11-26

## Overview

SAX-Meta 패키지에 Orchestrator와 skill-creator Agent를 추가하여 SAX 개발 워크플로우를 개선했습니다.

## Changes

### SAX-Meta

#### Added

- **orchestrator Agent**: SAX-Meta 요청을 분석하고 적절한 에이전트로 위임하는 Primary Router
  - Intent Classification & Routing 기능
  - Routing Decision Table 구현
  - SAX Core의 Routing-Only Policy 준수
  - 파일: `agents/orchestrator.md`

- **skill-creator Agent**: Skill 생성을 전담하는 대화형 Agent
  - 기존 skill-creator Skill을 Agent로 전환
  - 대화형 요구사항 수집 (What/When/How Complex)
  - Progressive Disclosure 자동 적용
  - 복잡도 기반 구조 설계 (Simple/Medium/Complex/Very Complex)
  - Anthropic Skills 표준 준수
  - 파일: `agents/skill-creator.md`

#### Removed

- **skill-creator Skill**: Agent로 전환되어 제거
  - 디렉토리: `skills/skill-creator/` 삭제
  - 복잡한 의사결정과 대화형 워크플로우가 필요하여 Agent가 더 적합

#### Changed

- **CLAUDE.md**: Agents 섹션에 orchestrator 및 skill-creator 추가
- **Skills 섹션**: skill-creator 항목 제거

## Impact

### SAX-Meta 개발자

- **워크플로우 개선**: SAX-Meta 작업 시 Orchestrator가 자동으로 적절한 Agent에 위임
- **Skill 생성 경험 향상**: skill-creator Agent의 대화형 인터페이스로 더 쉽게 Skill 생성
- **Progressive Disclosure 자동화**: 복잡도에 따라 자동으로 references/ 분리

### 주요 라우팅

| 요청 | 위임 대상 | 키워드 |
|------|-----------|--------|
| Agent 생성 | agent-creator | "Agent 만들어" |
| Skill 생성 | skill-creator | "Skill 만들어" |
| Command 생성 | command-creator | "Command 만들어" |
| 패키지 검증 | package-validator | "검증", "구조 확인" |
| 버전 관리 | version-manager | "버전", "릴리스" |
| 패키지 설계 | sax-architect | "구조", "설계" |

## Migration Guide

### 기존 skill-creator Skill 사용자

**Before** (v3.9.0):
```markdown
"Skill 만들어줘"
→ skill-creator Skill 직접 실행
```

**After** (v3.10.0):
```markdown
"Skill 만들어줘"
→ orchestrator가 skill-creator Agent에 위임
→ 대화형으로 요구사항 수집
→ Progressive Disclosure 자동 적용
```

### 변경 사항 없음

- `agent-creator`, `command-creator`, `sax-architect` Agent: 변경 없음
- `package-validator`, `version-manager` Skill: 변경 없음

## Technical Details

### orchestrator Agent

**Routing Decision Table**:
```yaml
Agent 생성: agent-creator
Skill 생성: skill-creator  # ← NEW
Command 생성: command-creator
패키지 검증: skill:package-validator
버전 관리: skill:version-manager
패키지 설계: sax-architect
```

### skill-creator Agent

**Progressive Disclosure Thresholds**:
| Total Lines | SKILL.md Target | Structure |
|-------------|-----------------|-----------|
| < 100 | ~100 lines | SKILL.md only |
| 100-200 | ~60-80 lines | + 1-2 refs |
| 200-300 | ~50-70 lines | + 2-3 refs |
| > 300 | ~50-80 lines | + 3-5 refs |

**Agent Workflow**:
1. Requirements Gathering (대화형)
2. Structure Planning (복잡도 분석)
3. Generation (파일 생성)
4. Validation & Guidance (다음 단계 안내)
