**SAX 개선 작업 시**:

1. **기존**: SAX 구조 변경 → 수동으로 Agent/Skill 추가 → 수동 버저닝
2. **새 방법**: "Semicolon AX 새 Agent 만들어줘" → sax-architect가 자동 처리
3. **변경 이유**:
   - SAX 메타 작업 전담 Agent 부재
   - 버저닝, CHANGELOG, 동기화 누락 위험
4. **주요 개선**:
