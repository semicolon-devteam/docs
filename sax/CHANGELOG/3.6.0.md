# SAX v3.6.0 - 2025-01-26

### Added

- **command-creator Agent** (SAX-PO)
  - SAX 슬래시 커맨드 생성 전문 에이전트
  - Claude Code slash command 규칙 준수 검증
  - 커맨드 파일 생성 및 구조 설계
  - CLAUDE.md 자동 업데이트 및 동기화 관리

- **create-command Skill** (SAX-PO)
  - SAX 커맨드 생성 가이드 제공
  - 네이밍 규칙 및 파일 구조 표준화
  - 이중 콜론 방지 (`/SAX::command` → `/SAX:command`)
  - 동기화 스크립트 제공

### Changed

- **CLAUDE.md** (SAX-PO)
  - Agents 섹션: command-creator 추가
  - Skills 섹션: create-command 추가

### Migration Guide

**새 커맨드 생성 방법**:

1. **Agent 활용** (권장):
   ```
   사용자: "새 SAX 커맨드 만들어줘"
   Orchestrator → command-creator Agent
   ```

2. **Skill 직접 사용**:
   ```bash
   # 1. 파일 생성 (: 프리픽스 없이)
   touch sax/packages/sax-po/commands/SAX/new-command.md

   # 2. 내용 작성
   # 3. CLAUDE.md Commands 섹션 업데이트
   # 4. .claude/ 동기화
   ```
