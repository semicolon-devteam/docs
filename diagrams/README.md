# 📊 협업 프로세스 다이어그램

이 디렉토리에는 협업 프로세스의 Phase별 상세 다이어그램이 포함되어 있습니다.

## 🎨 Excalidraw 다이어그램 편집 방법

1. [excalidraw.com](https://excalidraw.com) 접속
2. 해당 `.excalidraw` 파일을 드래그 앤 드롭
3. 다이어그램 편집
4. `File` → `Save to...` → 파일 저장
5. 변경사항 커밋 & 푸시
